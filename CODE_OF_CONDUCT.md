# Code Of Conduct

## Standard

This project follows the [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Expected Behavior

- Be direct and respectful.
- Focus criticism on code, design, documentation, and project outcomes.
- Assume public issues and pull requests may be read by future contributors.

## Reporting

Report unacceptable behavior through a private GitHub security advisory or by contacting the repository owner.

Maintainers may remove comments, close issues, or block participants who violate this standard.
